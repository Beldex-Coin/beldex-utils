## Installation

```bash
npm i @bdxi/beldex-paymentid-utils
```

## Usage

```js
const paymentTool = require('@bdxi/beldex-paymentid-utils');
const paymentID = 'f020853457bd110b';
const result = paymentTool.IsValidPaymentIDOrNoPaymentID(paymentID);
```
