> :warning: This project is under heavy development. Expect bugs & breaking changes.

## Installation

```bash
npm i @bdxi/beldex-locales
```

## Usage

```js
const {
  compatibleCodeFromLocale,
  mnemonicLanguages,
  supportedShortCodes,
} = require("@bdxi/beldex-locales");
```
